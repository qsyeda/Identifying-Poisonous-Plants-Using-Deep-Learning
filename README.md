
<h1>Identifying Poisonous Plants Using Deep Learning</h1>

<h2>Description</h2>
As part of the final project for our Deep Learning class, I developed three different deep learning models using various architectures and techniques like transfer learning to classify images of plants into poisonous and non-poisonous (binary image classification).
<br />


<h2>Python Libraries Uses</h2>
- <b>Tensorflow</b> 
- <b>Numy</b>
- <b>Pandas</b>
-<b>Python Imaging Library (PIL) </b> <i>*Note: This was used to verify if images were corrupted</i>
